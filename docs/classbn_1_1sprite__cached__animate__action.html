#include <bn_sprite_animate_actions.h>
template<int MaxSize>
bn::sprite_cached_animate_action class

Changes the tile set of a sprite_ptr when the action is updated a given number of times.

Template parameters
MaxSize Maximum number of sprite tile sets to store.

Contents

  • Reference
    • Public static functions
    • Public functions

This action differs from sprite_animate_action in that in this action all sprite tile sets to use are cached, so it is faster but takes way more VRAM.
